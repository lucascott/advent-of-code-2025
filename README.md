# Advent of Code 2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Go.

## Project Structure

```
cmd/        Entry points for each day
solutions/  Solution logic for each day
lib/        Shared utilities and helpers
data/       Input data files
```

## Running Solutions

Run a specific day's solution:

```bash
go run ./cmd/day01
go run ./cmd/day02
```

Or build and run:

```bash
go build -o aoc ./cmd/day01
./aoc
```
